Catchphrase REST API

An API built with Express and MongoDB. This is a REST API which can be accessed at the link below: 

					
	https://catchphrase-rest-api.herokuapp.com/catchphrases

Catchphrases

Get all the catchphrases 

					
	Type: GET
	Link: /catchphrases

Single Catchphrase

Get a catchphrases by id 

					
	Type: GET
	Link: /catchphrases/:id

Add Catchphrase

Create a new catchphrase 

					
	Type: POST
	Link: /catchphrases
	Body: 
		{
			"movieName": "AUSTIN POWERS: INTERNATIONAL MAN OF MYSTERY",
			"catchphrase": "Yeah baby, yeah!",
			"movieContext": "Austin Powers, conversing with his partner, Mrs. Kensington"
		}

Update Catchphrase

Update a catchphrases by id 

					
	Type: PATCH
	Link: /catchphrases/:id
	Body: 
		{
			"movieName": "AUSTIN POWERS: INTERNATIONAL MAN OF MYSTERY",
			"catchphrase": "Yeah baby, yeah!",
			"movieContext": "Austin Powers, conversing with his partner, Mrs. Kensington"
		}

	// All three fields are optional when updating

Remove Catchphrase

Delete a catchphrase by id 

					
	Type: DELETE
	Link: /catchphrases/:id
